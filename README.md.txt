# Booth Multiplier (Verilog HDL)

This project implements a 8 bit Booth Multiplier using Verilog HDL and with a modular datapath and FSM based Control path architecture 

## Features
- Booth algorithm based multiplication
- Separate datapath and control path design 
- FSM controlled iterative add/sub and arithmetic shift operation
- 16 bit final prodcut generated as {A,Q}
- Functional verification using a simulation testbench
- Tool used : Xilinx Vivado
